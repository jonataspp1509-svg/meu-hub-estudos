💻 Aprenda C++ na prática

Essa área ensina programação passo a passo. Você pode estudar pelo computador com VS Code ou pelo celular usando um compilador online. Depois de cada tópico, faça desafios e receba uma correção automática com nota e dicas.

⚙️ Como preparar o ambiente

💻 Opção 1: Computador com VS Code

📺 Tutorial completo (feito por mim):


▶️ Ver no YouTube
  1. Instale o Visual Studio Code.
  2. Instale um compilador C++.
  3. No VS Code, instale a extensão C/C++ da Microsoft.
  4. Crie um arquivo main.cpp.
  5. Compile e execute o programa.

Baixar VS Code

🧩 Compilador C++ no Windows

Uma opção comum é usar MSYS2/MinGW-w64 para instalar o GCC/G++.

  1. Instale o MSYS2.
  2. Abra o terminal MSYS2 UCRT64.
  3. Instale o GCC/G++.
  4. Teste com g++ --version.

Abrir MSYS2

🐍 Python também

Python é ótimo para começar lógica de programação, automação e scripts.

  1. Baixe o Python pelo site oficial.
  2. No Windows, marque “Add Python to PATH”.
  3. Teste no terminal com python --version.

Baixar Python

📱 Opção 2: Celular / sem instalar

Para quem está no celular ou não quer instalar nada, dá para usar o OnlineGDB.

  1. Abra o OnlineGDB.
  2. Escolha C++.
  3. Digite seu código.
  4. Clique em Run.

Programar no OnlineGDB
Importante: no celular, você não precisa instalar VS Code nem compilador. Use o OnlineGDB para treinar os desafios.
